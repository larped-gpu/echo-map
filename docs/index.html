How it works

  1. Chirp: LFM sweep through speaker (3 modes)
  2. Capture: 4-mic array at 48 kHz
  3. Process: matched filter, TOA range, DOA bearing, MFCC
  4. Classify: EchoNet CNN, 6 material classes
  5. Map: occupancy grid + material overlay
  6. Policy: pick next pose + chirp from map uncertainty
